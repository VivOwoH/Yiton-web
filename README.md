# 意通研达YitonTech

## 运行

- Next.js 14 + TypeScript
- Tailwind CSS

### 环境要求
- Node.js 18.0+
- npm 或 yarn

### 安装运行
```bash
git clone https://github.com/your-username/yitontech-website.git
npm install
npm run dev
npm run build
```

## 项目结构

```
src/
├── app/
│   ├── components/     # 页面组件
│   ├── globals.css     # 全局样式
│   └── page.tsx        # 首页
├── public/             # 静态资源
└── tailwind.config.js  # 样式配置
```
